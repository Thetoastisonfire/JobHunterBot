Run the bot!
Trigger a test run and wait for your first email
▼
  1. In your GitHub repo, click the Actions tab at the top
  2. Click Job Alert Checker in the left sidebar
  3. Click the Run workflow dropdown → green Run workflow button
  4. Wait about 30–60 seconds, then check your inbox
The first run emails everything matching your keywords from the past 3 days. After that, you only get an email when new jobs appear. The bot runs automatically at 9am and 5pm UTC every day.
